# WordUp - Приложение для изучения иностранных слов

Приложение для изучения иностранных слов на основе интервального повторения, разработанное на Flutter.

## Возможности

### Экраны авторизации
- **Экран входа** - форма входа с email и паролем
- **Экран регистрации** - форма регистрации с подтверждением пароля

### Основные экраны (после авторизации)

#### 1. Словари
- Список всех доступных словарей
- Отображение прогресса изучения для каждого словаря
- Информация о языке (от → до)
- Кнопка быстрого начала изучения

#### 2. Учить
- **Статистика**: 
  - Количество изученных слов
  - Количество повторений
  - Текущая серия дней подряд
- **Режимы обучения**:
  - Учить новые слова
  - Повторять слова
- Выбор словаря для изучения

#### 3. Настройки
- **Внешний вид**:
  - Переключение тёмной/светлой темы
  - Выбор языка интерфейса (Русский, English, Español, Deutsch)
- **Уведомления**:
  - Включение/выключение уведомлений о повторении
- **Интервалы повторения**:
  - Настройка интервалов между повторениями (в днях)
  - Использование алгоритма SM-2 для расчёта следующего повторения
- **Данные**:
  - Экспорт данных (в разработке)
  - Импорт данных (в разработке)
  - Сброс прогресса
- **Аккаунт**:
  - Выход из аккаунта

## Используемые технологии

- **Flutter** - фреймворк для кроссплатформенной разработки
- **Provider** - управление состоянием приложения
- **Material Design 3** - современный дизайн компонентов
- **SM-2 Algorithm** - алгоритм интервального повторения

## Запуск приложения

1. Установите Flutter SDK (версия 3.9.2 или выше)
2. Клонируйте репозиторий
3. Выполните `flutter pub get` для установки зависимостей
4. Запустите приложение: `flutter run`

## Зависимости

- `provider: ^6.1.1` - управление состоянием
- `cupertino_icons: ^1.0.8` - иконки для iOS стиля

## Интервальное повторение

Приложение использует модифицированную версию алгоритма SM-2:
- Оценка качества ответа от 0 до 5
- Расчёт коэффициента лёгкости (ease factor)
- Динамическое вычисление интервалов на основе настроек пользователя

Настроенные интервалы по умолчанию: 1, 3, 7, 14, 30 дней
